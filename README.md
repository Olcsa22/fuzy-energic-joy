# Fuzzy-energic-joy
Hello! This is a practice app for checking on my Spring Boot and Angular skills. 

## How to run the application

0. Visit the url [www.oliverexperiments.com](https://oliverexperiments.com/), so you can skip the rest of the steps

1. Clone the repository

2. Run the backend by opening it in a preferred IDE, then run the following command in folder `fuzzy-energic-joy`: `./gradlew build bootRun`

3. Change the current directory to `fuzzy-energic-joy-dashboard`

4. Run `npm i`

5. Run `npm run start`

6. After the command finishes, you can check the application on [http://localhost:4200](http://localhost:4200)
